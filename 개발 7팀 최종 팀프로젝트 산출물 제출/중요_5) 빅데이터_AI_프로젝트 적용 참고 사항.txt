1. 관리자 또는 사용자 웹사이트 중에서 서브 웹페이지는 Python Flask(또는 Django) 웹서버를 활용하시기 바라며,
   Beautiful Soup, Selenium, Scipy, Numpy, Pandas, Matplotlib, Seaborn 등 라이브러리를 활용 하셔서,
   웹스크래핑 후 검색된 웹크롤링 데이터를 OracleDB에 Insert하고, 웹페이지에서 활용하는 기능을 구현해 보기 바람.

2. 예를들어, 가격 정보를 웹 크롤링하여 사용자에게 정보를 제공하거나, 관심 있는 주제만 모아서 볼 수 있는 태그 검색 기능,
   이미지 또는 사진 클릭 시 종류를 알려주는 서비스, 음식 정보를 분석하고 본인 취향에 맞게 필터링해서 정보를 제공함,
   복용 중인 약품의 성분을 분석하여 보여주는 서비스 기능을 구현함

3. Beautiful Soup, Selenium 등을 활용해서, 이미지 크롤링을 통한 수집 데이터를
   Tensorflow 딥러닝을 통해 모델 학습 시킨 후, CNN 알고리즘을 적용한 이미지 분류 AI 구현

4. cx_Oracle, Oracle Client 등 활용하여 Oracle 11g DB 연동 CRUD 기능 구현

5. cx_Oracle, Oracle Client 등 활용하여 Oracle 11g DB 연동 후
   jQuery Mobile에서 로그인 회원과 내역 정보 확인 기능 구현

6. Python 기반 데이터 수집 및 분석 과정을 거쳐서 결과 예측 서비스 구현
   즉, 데이터 기반 결과 예정 정보 제공

7. 검색 이미지를 Django와 통신 후 selenium을 사용해서, 이미지 크롤링을
   통하여 수집한 데이터를 Tensorflow 기반으로 모델 학습 시킨 후
   CNN을 활용하여 검색 이미지 분류 기능 구현

8. 선택한 추천 여행 계획 확인 및 빅데이터 활용 분류 검색 구현
   (CNN, Sikit-learn, Django)

9. Google Cloud Vision 활용 이미지 검색 / 인식 및 분석 기능

10. Google STT / TTS + 파파고를 조합한 음성 및 문자번역 기능

11. Geocoder와 ODsay RestfulAPI를 이용한 길찾기 기능
