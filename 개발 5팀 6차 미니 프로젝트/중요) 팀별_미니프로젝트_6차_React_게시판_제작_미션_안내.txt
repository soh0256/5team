4. 요구 사항 분석
   1) 요구 적용 기술

      ① Front-End 웹사이트 = React

      ② Back-End 웹사이트 = Express, Node.js, Oracle11g DB,
                                     회원 가입 시 주소 정보 입력은 "DAUM 우편번호, KAKAO 도로명주소 적용 API" 연동하여,
                                     Oracle11g DB에 주소 정보값들이 저장될 수 있도록 하시기 바랍니다. 또한,
                                     "네이버 아이디로 로그인"을 희망할 경우, 가능하도록 기능을 구현 하시기 바라며,
                                     이때, Naver 아이디 연동 정보는 "이름", "이메일" 2가지 칼럼값을 Naver 웹사이트와 데이터 연동하시기 바랍니다.
                                     그리고, Naver 웹사이트에서 가져온 "이름", "이메일" 2가지 칼럼값은 Oracle11gDB에 저장하도록 처리해 주시기 바랍니다.
 
  2) 개발 요구 기능 : [필수 기능 개발 요구 사항]

     ① 로그인, 로그아웃

     ② JWT(JSON Web Token) 활용 회원 로그인 정보 암호화 처리
   
     ③ "네이버 아이디로 로그인"을 희망할 경우, 가능하도록 기능을 구현 하시기 바라며,
        이때, Naver 아이디 연동 정보는 "이름", "이메일" 2가지 칼럼값을 Naver 웹사이트와
        데이터 연동하시기 바랍니다. 그리고, Naver 웹사이트에서 가져온 "이름", "이메일"
        2가지 칼럼값은 Oracle11gDB에 저장하도록 처리해 주시기 바랍니다.

     ④ 회원가입(프로필 이미지 업로드, 아이디, 비밀번호, 이메일, 성명, 연락처,
        주소, 상세주소, 전체 약관 동의 "전체 동의" 체크, 서비스 약관 동의 "필수" 체크,
        약관보기, 마케팅 수신 동의 "선택" 체크, 약관보기 등 포함), 회원탈퇴

     ⑤ 회원 가입 시 주소 정보 입력은 "DAUM 우편번호, KAKAO 도로명주소 적용 API"
        연동하여, Oracle11g DB에 주소 정보값들이 저장될 수 있도록 하시기 바랍니다.
  
     ⑥ 아이디 찾기(회원 가입 되어 있는 "이름", "휴대폰번호" 를 입력하고 "찾기" 버튼을
         클릭하면, Alert 메소드로 회원 가입 후 최종 DB 저장 "아이디"를 알려줍니다.
         => Alert 메소드 알림 메시지 예시 : "아이디는 jangnara 입니다."
 
     ⑦ 비밀번호 찾기(회원 가입 되어 있는 "이름", "휴대폰번호", "아이디"를 입력하고
         "찾기" 버튼을 클릭하면, Alert 메소드로 회원 가입 후 최종 DB 저장 "비밀번호"를
         알려줍니다. => Alert 메소드 알림 메시지 예시 : "비밀번호는 1234 입니다."
 
     ⑧ 게시판 글쓰기(이미지 등록 포함), 게시판 글수정, 게시판 글조회,
         게시글 상세보기(제목 클릭하면 볼수 있게 함), 게시판 글삭제,
         웹사이트 하단 페이징 처리(1, 2, 3, ...)

     ⑨ 게시판 작성글에 대해 "좋아요" 체크 및 저장 가능하도록 함.
  
     ⑩ 게시판 작성글에 대해 별평가(별은 5개 내외로 하고,
        별 아이콘을 선택해서 평가할 수 있도록 함) 및 저장 가능하도록 함.
